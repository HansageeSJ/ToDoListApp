# Angular + .NET ToDoListApp (In-Memory)

A simple ToDoListApp built with Angular 18 and .NET 8 Web API. Data is stored in-memory on the server (no database).

## Requirements
- .NET 8 SDK
- Node 20+ and npm
- Angular CLI (`npm i -g @angular/cli`)

## Run

### Backend

cd ToDoListApp_Api
dotnet restore
dotnet run

### Frontend

cd todo-ui
ng serve
