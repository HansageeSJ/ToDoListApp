# Angular + .NET TODO (In-Memory)

A simple TODO list app built with Angular 18 and .NET 8 Web API. Data is stored in-memory on the server (no database).

## Requirements
- .NET 8 SDK
- Node 20+ and npm
- Angular CLI (`npm i -g @angular/cli`)

## Run

### Backend
```bash
cd TodoApi
dotnet restore
dotnet run
